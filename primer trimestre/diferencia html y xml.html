diferencias entre html y xhtml xhtml es compatible con xml ya que html no cumple todos los requesitos que debe tener para ser lenguaje de marcas. Todas las etiquetaas y atributos deben ir en minuscula. Todos los elementos deben cerrarse. incluso los elementos vacios. no se permiten errores de alimidamiento. es decir las etiquetas no deben solaparse. todos los valores de atributos deben estar entre comillas simples o dobles. todos los atributos deben tener un valor.
